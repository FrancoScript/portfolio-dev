# Portfolio Dev

Portfolio personal hecho con Astro y Tailwind, mostrando proyectos web y microproyectos de práctica.

## Tecnologías

- React
- Astro
- Tailwind CSS
- HTML / CSS / JavaScript

## Proyectos

1. **Sitio web para negocio de Hojalatería**  
   Página informativa para empresa de hojalatería y fabricación de extractores eólicos. Diseño responsive, optimización visual y pequeñas animaciones.  
   _Ver en portfolio_

2. **Chuck Norris Quote Generator**  
   Microproyecto que consume API pública para mostrar frases aleatorias. Incluye diseño responsive, animaciones y manejo de eventos con JavaScript.  
   _Ver en portfolio_
